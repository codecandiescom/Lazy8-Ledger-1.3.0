SQL Report
This report generator will allow you to generate any type of report you can dream up.  But there is one catch, you will almost need to be a professional programmer to use it.  That is, you must be proficient in SQL programming.  However, there are several reports already done that you can choose from in the convienient drop down list box.  You will need a definition of how the database tables are setup, see technical specification
 

Report

This is where you can choose an already prepared report definition.

SQL Query window

This is where you will type in the report definition if you are good at SQL programming. 

Save button

After you have entered your report definition.  You can press this button to save it and the name will be the same as what is showing in the REPORT field named above.  You may want to use the Save as button described below instead.

Save as button

When you have written a new report definition, then you can press here and then you will be asked to give a name for the report which will later show up in the REPORT field described above.

Delete button

Be very careful here, this will delete the report showing in the REPORT field described above.

Clear button

This will clear the SQL Query window described above.

Get Report button

Press this button when all the previous fields are entered.  The report will appear in a seperate word processing window

Get report into clipboard Button

Press this button when all the previous fields are entered.  The report will be directly put onto the clipboard. You must then paste the report into some other wordprocessor or spreadsheet. If you paste into a wordprocessor, then you will attain the best results if after you have pasted into the wordprocessor, that you select the entire report and then select -convert to table- in the wordprocessing program

Exit

Exit this screen
