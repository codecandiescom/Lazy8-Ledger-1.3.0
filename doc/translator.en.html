Translator

This module allows you to change every text that appears anywhere (except help) in this program to any language or to personalize this program to suit just you.  Note that if you do use this module to adjust this program to another language than I hope you email to me the Lang.??.bin file which will be created when you are finished making the changes.  The "??" are the 2 digit language code for the language you produced.

This program is 100% unicode compatible so any language in the world can be used here.

There are two columns so that you may compare your translations with another language.  Both columns operate identically so only one will be described below.

So that your changes in the language will show up in this program, you must save your changes with the save button and then change to another language and then back again to the language you just made changes to. Then restart the program.

Language

Here you choose the language you wish to edit.

Load button

Then, you press this button to load the language.  If such a language has never been created before, then you will need to go to the NEW button.

New button

This will create a new language.  The language that actually appears is english which you need to change to your own language.

Cancel button

Abort any changes to the language you have made and exit the screen.

Help button

Show the help screen associated with this form.

Save button

You must press this button to save your work or it will be lost
