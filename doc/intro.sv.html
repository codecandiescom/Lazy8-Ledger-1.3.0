Hej och välkommen till Lazy8 Bokföring!  Jag hoppas att du kommer att ha lika roligt då du använder det här programmet som vi hade då vi producerade det åt dig.

Vem kan använda detta program

Detta bokföringsprogram är framtaget för personer som har erfarenhet av bokföring.  Om du inte har erfarenhet av bokföring behöver du gå till bokhandeln och skaffa dig en väldigt liten och enkel bok om bokföring (dubbel bokföring) OCH du behöver ha en redovisningskonsult som hjälper dig se till att du använder detta program på rätt sätt.  Detta är ett mycket kraftfullt program, men det finns inte så mycket kontroll på vad du gör. Därför är det lätt att göra misstag som ger dig ett felaktigt resultat.  Egentligen är det här programmet mest lämpat för dem som kan göra sin bokföring för hand men som hellre vill ha ett program som automatiserar processen och som till och med  kan utföra några av de mer komplexa beräkningar som normalt redovisningskonsulten brukar göra.


Vad kan programmet erbjuda

Det här programmet kommer att ge dig allt du behöver för bokföring i vilket företag som helst eller för privat användning.  Du kan använda vilken typ av bokföring som helst och bryta upp aktiviteterna i din bokföring på hur många konton som helst.  Det finns flera färdigdesignade rapporter som motsvarar de grundläggande behoven inom bokföring och så finns det verktyg för att göra dina egna rapporter. 

Dessutom är detta program 100% unicode compatible på alla tänkbara sätt så du kan ändra vilken text som helst varhelst den dyker upp i detta program till vilket språk som helst på denna planet. 

För närvarande är detta program bara tillgängligt på engelska och svenska för det är de enda språk jag kan, fastän när du bläddrar igenom det här programmet kanske du börjar undra om jag kan något språk över huvud taget....Andra språk kommer snart att vara tillgängliga ju mer vänliga användare kommer att använda detta program och skicka mig deras översättningar.
