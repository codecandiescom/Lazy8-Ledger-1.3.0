Create new company
This is just an easy way to start bookkeeping on a new company. The same result can almost be had by going to DataEntry->Company and then to DataEntry->Accounts. This however also adds the capability to load up a default EU BAS 2000 account plan. Again, that can also be done by restoring a default account plan with Advanced->backup. This does everything in one easy proceedure.
